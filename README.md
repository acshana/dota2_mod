## 项目背景

1. steam基本不会允许盗号饰品的找回，问就是会导致通货膨胀，为了其他玩家的饰品保值，淦你老湿。
2. 目前在用3DM @ラケシス 老师的mod找回一些饰品的感觉，但是每次都得拷贝来拷贝去的很麻烦，因此根据ラケシス 老师的教程图片做了个简单的脚本

## 如何使用

1. 下载本本项目中的dota2_mod_ansi.bat脚本，比如到桌面
2. 使用win+R键打开运行
4. 输入cmd回车打开命令行工具
5. 运行脚本（比如脚本在桌面）：
cd Desktop
dota2_mod_ansi.bat
